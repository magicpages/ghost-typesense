Search UI playground

Loads the built /search.min.js the way Ghost does, against a mocked Typesense backend — so you can drive the widget and every feature without a Ghost install or a real Typesense server. Press / or Cmd/Ctrl + K, or click Open search.

Configuration

Applying re-creates the widget with the chosen config. Leave the endpoint blank to search the built-in mock dataset (try “tomato”, “ghost”, “garden”).
